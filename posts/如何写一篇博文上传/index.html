如何写一篇博文上传？

撰写博文上传私人网站教程

第一步

安装hugo与github

第二步

创建站点

在hugo文件夹输入 hugo new site 站名,生成新站点（如myblog）。 congratulation提示创建成功，出现站站点文件夹（myblog）。

创建文章

cd到站点文件夹（myblog），输入hogo new post\文章标题.md,将在post文件夹创建一篇文章。

下载网站主题

hugo官网themes选择。下载格式，解压到themes文件夹，将后缀-master删除，进入主题文件夹exampleSite，将文件复制粘贴到站点文件夹（myblog）内，替换原文件。

建设主题

输入hugo server -t 主题文件名 --buildDrafts,即可进入网站预览，访问地址为localhost：1313 #hugo server -t hugo-nederburg-theme –buildDrafts

第三步

注册登陆gihub

创建新仓库

仓库地址为 账号名.github.io

上传本地网站

1.输入hugo --theme=主题文件名 --baseUrl="https://github仓库地址" --buildDrafts ,在本地生成public文件夹，文件为已渲染网页文件。
#hugo –theme=hugo-nederburg-theme –baseUrl="https://onepassone.github.io.git” –buildDrafts 2.cd至public文件夹，创建.git文件夹，即为git本地仓库。

或用git init也可以创建。
3.输入git add .,将所有文件提交到本地git仓库。
4.输入git commit -m "提交说明"
5.输入git remote add origin https://github.com/账号名/账号名.github.io.git，与远程仓库关联，成功无提示。 #git remote add origin https://github.com/onepassone/onepassone.github.io.git 6.输入git push -u origin master,将文件上传至远程仓库。
7.刷新网页即可看到github有文件在内，至此网页创建完毕。
